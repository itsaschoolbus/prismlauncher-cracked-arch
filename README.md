## Automated [ayugram-desktop](https://aur.archlinux.org/packages/ayugram-desktop) package repo for Arch Linux
Builds are automated using Actions.

You can use it by adding this to `pacman.conf`:
```
[ayugram]
SigLevel = Never
Server = https://itsaschoolbus.github.io/$repo-arch
```

Inspired by the [materialgram repo](https://github.com/materialgram/arch).
